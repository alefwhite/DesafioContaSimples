# DesafioContaSimples
Processo Seletivo Conta-Simples / O que deve ser desenvolvido: Se você tem familiaridade com backend, o desafio consiste em  desenvolver uma aplicação que disponibiliza uma API REST onde retorna um conjunto de informações a seu critério. Fique a vontade para utilizar a stack que você mais tem familiaridade. Você pode consultar os dados de uma tabela ou exibir os dados mockados.  ·  Se você tem familiaridade com frontend, o desafio consiste em criar uma Landing Page sobre um tema aberto, onde você tem familiaridade e conhecimento e gostaria de compartilhar com a gente. (Pode utilizar a stack que você tem familiaridade) - Você pode escolher entre o desafio frontend e backend.
